# Archivo histórico — datos/

Ficheros obsoletos del formato de carga inicial del plan de Relevo
Generacional. Conservados como referencia histórica.

- `KPI.xlsx` — hoja original de indicadores y valores anuales.
- `Seguimiento_proyecto.xlsx` — hoja original de ámbitos, actuaciones y
  bitácora de seguimiento.

El formato actual está en la carpeta superior (`datos/`) bajo el nombre
`Plan_Sectorial_<NOMBRE>.xlsx` y se documenta en
`Guia_de_carga_de_planes.docx`. Se carga con
`scripts/importar_plan.py [--reemplazar]`.
